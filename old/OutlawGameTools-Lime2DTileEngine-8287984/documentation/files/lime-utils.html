utils:clampPosition (x, y, bounds)
Clamps a position to within bounds.

Parameters

  • x: The X position to clamp.
  • y: The Y position to clamp.
  • bounds: The bounding box for the clamping. A table with x, y, width and height. X and y are top left corner.

Return values:

  1. The clamped X position.
  2. The clamped Y position.
utils:convertFramesToTime (frameCount, fps)
Converts a frame count into minutes, seconds and tenths. Can be used to display a timer like this -- minutes .. ":" .. seconds .. ":" .. tenths

Parameters

  • frameCount: The count of the frames
  • fps: The current fps. Either 30 or 60, will default to 30.

Return values:

  1. minutes Minute value.
  2. seconds Second value.
  3. tenths Tenths of second value.

Parameters
