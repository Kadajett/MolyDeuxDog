TileSet:new (data, map, firstgid, rootDir)
Create a new instance of a TileSet object.

Parameters

  • data: The XML data.
  • map: The current Map object.
  • firstgid: The first GID of the tileset. Optional and only needed if different from the TMX data. Used for custom maps.
  • rootDir: The root dir of the tileset image. Optional.

Return value:

The newly created TileSet instance.
